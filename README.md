# EDA & Visualization Projects

This repository contains some of the EDA and data visualization work I’ve done while learning data analysis.

I originally built these projects a few months ago, and recently organized them properly into this repository.

---

## What’s inside

### Amazon Sales Analysis

Worked on a sales dataset to understand how different categories perform and how sales change over time.
Focused on basic cleaning and then explored trends using graphs.

### Ecommerce Data Cleaning

This project is mainly about cleaning a messy dataset — handling missing values, fixing formats, and making it usable for analysis.

### GDP vs Population Analysis

Tried to compare GDP and population data to see if there’s any visible pattern or relationship between them across countries.
